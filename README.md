# 📊 Cellphone Market & Consumer Behaviour Analysis

## 🚀 MBA Data Analytics Mini Project

An end-to-end **consumer behaviour and mobile market analysis project** built using **Python, Tableau, Statistical Analysis, and Data Visualization**.

This project focuses on understanding:

* 📱 Mobile phone pricing behaviour
* 👥 Consumer purchase patterns
* 💰 Salary vs spending trends
* 📸 Hardware specification impact on pricing
* 🏙️ Regional and demographic market behaviour

The project combines:

* **Exploratory Data Analysis (EDA)**
* **Statistical Hypothesis Testing**
* **Interactive Tableau Dashboards**
* **Business Insight Generation**

# 💡 Major Business Insights

## 📍 1. Price is Driven by Specifications

RAM and internal memory showed the strongest relationship with price.

## 📍 2. iOS Devices Command Premium Pricing

Statistical testing confirmed significant pricing differences between Android and iOS.

## 📍 3. Camera Megapixels Do Not Always Define Premium Quality

Flagship devices focus more on sensor optimization than raw megapixel count.

## 📍 4. Battery Size Has Become Standardized

Most smartphones are concentrated around similar battery capacities.

## 📍 5. Consumer Demographics Have Limited Impact on Hardware Preferences

Age and occupation had minimal influence on device specifications.
